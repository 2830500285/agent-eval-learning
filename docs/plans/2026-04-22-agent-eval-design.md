# Agent Eval Learning Design

## Goal

Create a bilingual GitHub knowledge base focused on agent and LLM evaluation, with an initial emphasis on major labs, respected practitioners, and actionable tooling.

## Scope for v1

- bilingual Markdown-first structure
- curated high-signal links instead of copied source content
- overview, company notes, practitioner notes, tooling map, reading list, and one reusable template

## Out of scope for v1

- full paper-by-paper coverage
- automated site generation
- dataset mirroring
- benchmark result reproduction

## Information architecture

- `docs/00-overview`: mental models and principles
- `docs/01-companies`: OpenAI, Anthropic, Google Research / DeepMind
- `docs/02-practitioners`: Hamel Husain, Eugene Yan, Shreya Shankar
- `docs/03-tooling`: initial eval tool landscape
- `docs/04-templates`: reusable eval specification template
- `docs/05-reading-list`: concise recommended reading order

## Editorial rules

- prefer primary sources
- summarize for builders
- keep every major page bilingual
- optimize for extensibility over exhaustiveness
