# Phoenix / Phoenix

## English

Arize Phoenix is valuable when you want open-source friendly observability and evaluation workflows for LLM applications.

### What Phoenix is good at

- local or self-managed workflows
- trace inspection
- prompt and response analysis
- evaluator-based quality workflows

### When to choose Phoenix

- you want an open-source leaning stack
- you need observability first and evals second
- you want to avoid deep lock-in early

### Sources

- https://arize.com/docs/phoenix
- https://arize.com/docs/phoenix/evaluation
