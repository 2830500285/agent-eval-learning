# Starter Reading List / 起步阅读清单

## English

### Must-read first

1. OpenAI Evaluation best practices
   - https://developers.openai.com/api/docs/guides/evaluation-best-practices

2. Hamel Husain and Shreya Shankar: LLM Evals: Everything You Need to Know
   - https://hamel.dev/blog/posts/evals-faq/

3. Eugene Yan: Task-Specific LLM Evals that Do and Don't Work
   - https://eugeneyan.com/writing/evals/

### Benchmark-oriented

4. OpenAI: Introducing SimpleQA
   - https://openai.com/index/introducing-simpleqa/

5. OpenAI: BrowseComp
   - https://openai.com/index/browsecomp/

6. Anthropic: Model-Written Evaluations
   - https://www.anthropic.com/research/discovering-language-model-behaviors-with-model-written-evaluations

### Safety and robustness

7. Anthropic: Forecasting rare language model behaviors
   - https://www.anthropic.com/research/forecasting-rare-behaviors

8. Anthropic: Sabotage evaluations for frontier models
   - https://www.anthropic.com/research/sabotage-evaluations

9. OpenAI: TruthfulQA
   - https://openai.com/index/truthfulqa/

### Tooling

10. OpenAI simple-evals
    - https://github.com/openai/simple-evals

11. Promptfoo docs
    - https://www.promptfoo.dev/docs/intro/

12. Langfuse evaluation overview
    - https://langfuse.com/docs/evaluation/overview

13. LangSmith evaluation quickstart
    - https://docs.langchain.com/langsmith/evaluation-quickstart

14. Vertex AI evaluation overview
    - https://cloud.google.com/vertex-ai/generative-ai/docs/models/evaluation-overview

15. Microsoft Azure evaluation concepts
    - https://learn.microsoft.com/azure/ai-foundry/concepts/evaluation-evaluators
